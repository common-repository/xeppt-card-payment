A WordPress plugin to make payments via XEPPT Card. 

== Description ==

This plugin is useful for the XEPPT Card Holders to make payments on WooCommerce. 

== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
  Step 1: Install the plugin and activate it.
  Step 2: Go to Payments tab in WooCommerce Settings and then to Manage XEPPT Card Payment.
  Step 3: Fill all the required fields and save.
  Step 4: Now go to website and add required product(s) in the cart.
  Step 5: On the checkout page you will find the XEPPT Card Payment option. Here fill the Card Holders Name and XEPPT Card Number to make payment.
  
= How to uninstall the plugin? =
  
Simply deactivate and delete the plugin.
